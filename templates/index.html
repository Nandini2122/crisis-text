What Crisis Text Does

🌊

Disaster Message Analysis

Processes raw, unstructured disaster messages using NLP techniques including tokenization, TF-IDF vectorization, and semantic analysis to extract structured emergency data.

⚠️

Emergency Severity Detection

Determines the level of emergency — High or Medium — using a trained Random Forest classifier with 78.8% accuracy on real disaster message data.

📍

Location Detection

Extracts the geographic location from disaster messages using substring matching against 801 Indian districts and cities across all states and union territories.

📞

Helpline Recommendation

Provides the appropriate state-level SDMA disaster helpline number from a verified database of 84 helplines across all Indian states and union territories.

Project Overview

About Crisis Text

Understanding the problem, the solution, and the technology behind India's AI-powered disaster message analysis platform.

🎯

Problem Statement

During disasters, thousands of people post urgent messages on social platforms requesting immediate help. These messages are unstructured, scattered across platforms, and often difficult to prioritize. Emergency responders face an impossible task — manually reading and routing hundreds of messages under extreme time pressure, leading to delayed response and preventable loss of life.

💡

Our Solution

Crisis Text uses Natural Language Processing and a trained Random Forest machine learning model to automatically parse incoming disaster messages. The system identifies the emergency severity level, extracts geographic location, determines the type of help needed, and instantly recommends the correct disaster helpline — all in real time. This converts raw, chaotic text into structured, actionable emergency intelligence.

⚙️

Technologies Used

The system is built on a modern Python-based NLP stack, combining classic machine learning with linguistic preprocessing for accurate, fast predictions.

Python 3.12 scikit-learn TF-IDF Vectorization Random Forest NLTK spaCy rapidfuzz pandas Flask / FastAPI HTML + CSS + JS
